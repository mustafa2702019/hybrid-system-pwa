# Hybrid System: Ayanokoji × Jinwoo

Mobile-first cinematic PWA for the 12-Month Hybrid Plan (2-4-6), built with:

- React + TypeScript
- TailwindCSS
- Framer Motion
- Dexie (IndexedDB, offline-first)
- Zustand
- Recharts
- Vite PWA (service worker + manifest)

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

```bash
npm run deploy
```

## Install as PWA

- Open the deployed app in Chrome/Edge.
- Use Install App / Add to Home Screen.
- After first load, the service worker caches assets for offline use.
